# История изменений

## 1.0.2 - 2026-04-27

- Репозиторий переведен на русский язык по умолчанию.
- Добавлен отдельный английский файл `README.en.md`.
- Добавлен `CHANGELOG.md`.
- Обновлен release archive, чтобы внутри были русская и английская документация.

## 1.0.1 - 2026-04-27

- Приложение скрыто из Dock и Cmd-Tab через `LSUIElement=true`.
- Installer теперь принудительно выставляет `LSUIElement=true` после копирования `.app` в `/Applications`.

## 1.0.0 - 2026-04-27

- Первый рабочий релиз.
- Фоновый AppleScript-хук отслеживает `~/.codex/transcription-history.jsonl`.
- При русской раскладке отправляет физический `Cmd+V` через `key code 9`.
- Добавлены install/uninstall-скрипты и готовый `.app` в release zip.
